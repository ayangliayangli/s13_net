# 流程
        新建三个用户 John Liz Peter
        John fall in love with Liz
        John break love with Liz
        John work hard -- 80%的时间在努力的工作
        Peter work -- 20% 的时间在努力的工作
        如果现在Peter 和 John 的资产差不多,John说一句话
        否则, 逆袭失败
        
# 亮点
        使用随机数的概念,预测101次工作轮次后John 和 Peter 的财务状况
