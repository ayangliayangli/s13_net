* s13-net 是后来（20160822）才规划的，所以前5天的划分不准,以day-xx 里面的分md为准

* 这里的每周基本相当于线上的每天

# 理想目录设计架构
* bin === 程序入口
* src === 程序主要逻辑
* lib === 通用的模块,类似common
* config === config directory
* db === data directory
* log ==== log directory
